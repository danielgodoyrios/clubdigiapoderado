# CLUBDIGI — App para apoderados

> Gestión del pupilo deportivo: asistencia, pagos, comunicados y carnet.

## Stack
- **Framework**: Expo SDK 52 + React Native 0.76
- **Color primario**: `#1A3A7C` (azul ClubDigi)
- **Navegación**: 3 tabs — Inicio / Agenda / Gestión

## Inicio rápido

### Windows
```
install.bat
npm start
```

### Mac / Linux
```bash
chmod +x install.sh && ./install.sh
npm start
```

## Concepto

ClubDigi es la app **exclusiva para apoderados**. Permite:
- Ver el carnet del pupilo (acceso directo desde header)
- Seguir asistencia en tiempo real
- Pagar cuotas del club
- Recibir y firmar comunicados
- Ver agenda de entrenamientos y partidos

## Carnet en header
Solo icono + punto verde. Al tocar → carnet del pupilo con QR y token.
